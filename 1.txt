Настройки подключения
Введите webhook url
Адрес, на который отправлять уведомления
Укажите в этом поле адрес вебхука, который будет устанавливаться в ваш филиал при подключении приложения
Введите callback url
Callback Url
Укажите url адрес, на который вы будете получать вебхуки об отключении интеграции на стороне YCLIENTS
Введите registration redirect url
Registration Redirect Url
Укажите адрес формы регистрации в вашем сервисе, на который пользователь попадет после нажатия на кнопку "Подключить" в карточке вашего приложения
Открывать форму регистрации в iframe
При нажатии на кнопку "Подключить" страница регистрации вашего сервиса будет открываться в табе "Настройки" в карточке вашего приложения в маркетплейсе. Подробнее
Передавать данные пользователя при подключении интеграции
При подключении интеграции, вам будут передаваться данные пользователя. Вы сможете автоматически зарегистрировать пользователя в своем сервисе, не заставляя его заполнять форму. Подробнее

При подключении интеграции пользователю будет предложено выбрать филиалы. Так, он сможет сразу подключить интеграцию во все свои филиалы за один раз. Подробнее
